## Tool Details
- **Name:** Pagewell
- **Repo:** ben-gy/pagewell (private)
- **Category:** document-tools
- **Audience:** Non-technical people handling sensitive PDFs (legal / medical / financial / HR) who don't want to upload to a "free online PDF tool"
- **Stack:** Vite 6 + vanilla TypeScript + Vitest
- **Browser APIs:** pdf.js (pdfjs-dist), pdf-lib, Web Workers, Canvas 2D, Drag and Drop, File System Access, Web Share, Service Worker (PWA)
- **Worker strategy:** pdf.js parses in its bundled worker; a dedicated `assemble.worker` runs pdf-lib to build the output PDF off the main thread with per-page progress

## Privacy Model
- **Protected:** PDF contents and the output file — parsed, previewed and reassembled entirely in the tab; no upload endpoint; works offline after first load
- **Not protected:** nothing is transmitted at all; pdf-lib preserves any hidden per-page metadata (pair with metascrub)
- **Trust surface:** static bundle over GitHub Pages TLS + the browser's own PDF/canvas engines; no third-party servers, CDNs, fonts, trackers or analytics

## Architecture Decisions
- Vanilla TS (single view; a small store + minimal grid re-render with cached canvases is lighter than React)
- pdf-lib for lossless page copy (never re-encodes page content — thumbnail fidelity is irrelevant to the exported bytes)
- Output assembly in a dedicated worker; thumbnails cached per page id so grid re-renders are instant

## Bugs Caught & Fixed During Verification
1. **pdf.js thumbnail rendering hung** in backgrounded/headless tabs — pdf.js schedules canvas paint chunks via `requestAnimationFrame`, which doesn't fire when the tab isn't painting. Fixed by rendering with `intent: 'print'` (microtask scheduling instead of rAF). Also a genuine real-world robustness fix for backgrounded tabs.
2. **Export buttons stayed disabled after import** — `importOne` set `busy=true`, `addDoc` rendered the toolbar disabled, and `setBusy(false)` didn't re-render. Fixed so `setBusy` re-renders when pages exist.
3. **Hardened the keyboard handler** — guarded the `.matches` call against non-Element event targets so shortcuts can't be swallowed.
