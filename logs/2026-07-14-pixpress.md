## Tool Details
- **Name:** Pixpress
- **Repo:** ben-gy/pixpress
- **Category:** image-tools
- **Audience:** Broad — parents emailing photos, shop owners fitting upload caps, developers
  converting PNG→WebP/AVIF for the web. Anyone who'd rather not upload images to a stranger's server.
- **Stack:** Vanilla TypeScript + Vite 6 + Vitest
- **Browser APIs:** WebAssembly (@jsquash MozJPEG/WebP/AVIF/OxiPNG), createImageBitmap +
  OffscreenCanvas, Web Workers, Transferable ArrayBuffer, Clipboard, Web Share, Service Worker (PWA)
- **Worker strategy:** Single dedicated worker with a decoded-bitmap cache — decode once, then
  re-encode instantly on any quality/format/size change (falls back to a fresh decode on cache-miss).

## Privacy Model
- **Protected:** every pixel and byte of every image, all EXIF/GPS metadata (dropped on
  re-encode), filenames — none leave the tab.
- **Not protected:** initial page + wasm load from GitHub Pages' CDN sees the visitor's IP (never
  images); a cookie-less Cloudflare Web Analytics beacon records an anonymous page view.
- **Trust surface:** the pinned static bundle, the TLS chain to GitHub Pages, and the @jsquash
  WASM shipped inside the bundle — no third-party runtime CDN.

## Architecture Decisions
- **@jsquash over native `canvas.toBlob`:** the whole point of a *compressor* (vs "Save As") is
  meaningfully smaller files. MozJPEG/WebP/AVIF/OxiPNG beat the browser's built-in encoders and
  justify the tool's existence. Codecs are dynamically imported per format so a heavy/failed
  codec never blocks the others.
- **AVIF mapping:** @jsquash/avif has no `quality` option — mapped the 0–1 UI quality onto its
  inverted `cqLevel` (0 best … 63 worst) so the slider actually does something. Caught during
  API inspection before build.
- **Decode-once cache:** keeps the tool snappy (Squoosh-like) when experimenting with settings.
- **PWA runtime-cache for wasm:** switched from precaching all wasm (8.5 MB up front) to
  on-demand CacheFirst — first paint precache dropped to 333 KB while staying offline-capable.
- **OxiPNG guarded:** wrapped in try/catch so PNG output still succeeds if the (MT) optimiser
  can't init without cross-origin isolation on GitHub Pages.

## Errors & Resolutions
- **AVIF quality ignored:** @jsquash/avif takes `cqLevel`, not `quality`; a loose cast would have
  silently produced constant-quality AVIFs. Fixed by mapping quality → cqLevel before shipping.
- **PWA over-precaching:** initial config precached 8.5 MB of wasm. Switched wasm to on-demand
  runtime caching; precache now 333 KB.
